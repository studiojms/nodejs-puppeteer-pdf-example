# nodejs-puppeteer-pdf-example

Create PDF using Puppeteer

## Installing dependencies

To install the dependencies, run:

```
npm install
```

## Executing the project

To execute the project, run:

```
npm start
```

### How it works

When the project is running, inform a URL to be converted to PDF in the query param `target`, as follows:

```
http://localhost:4000/pdf?target=https://google.com
```

### Example showing chart in pdf

```
http://localhost:4000/pdf?target=https://h9zif.csb.app/
```
